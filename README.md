# 🃏 欢乐斗地主 v4.0

一款基于纯前端技术的网页斗地主游戏，支持单机对战AI + 局域网多人对战，完全离线可用。

## ✨ 特色功能

- **单机模式** — 快速开始，与2个AI对战
- **多人模式** — 同一局域网（同浏览器多标签页）创建/加入房间
- **智能AI** — 优化出牌策略：飞机/顺子/三带/炸弹保护/队友配合
- **托管模式** — 一键托管，AI自动代打
- **语音播报** — 牌型专属语音（顺子、飞机、炸弹、王炸等）
- **音效系统** — Web Audio API 合成，完全离线，背景音乐+出牌音效
- **炸弹特效** — 屏幕震动+爆炸闪光
- **胜利彩带** — Canvas 粒子烟花庆祝
- **无牌自动不出** — 没有牌能打过时自动跳过

## 🎮 牌型支持

| 牌型 | 说明 |
|------|------|
| 单张 | 任意一张牌 |
| 对子 | 两张相同的牌 |
| 三张/三带一/三带二 | 三张相同 + 可带牌 |
| 顺子 | 5张以上连续单牌 |
| 连对 | 3对以上连续对子 |
| 飞机 | 两组以上连续三张（可带翼） |
| 炸弹 | 四张相同 |
| 王炸 | 大王+小王（最大） |

## 🚀 本地运行

无需安装任何依赖，直接用浏览器打开即可：

```bash
# 方式1：直接打开
open index.html

# 方式2：本地HTTP服务器（推荐，支持多标签页多人对战）
npx serve .
# 或
python3 -m http.server 8080
```

然后在浏览器访问 `http://localhost:8080`

## 🛠 技术栈

- **HTML5 + CSS3 + 原生JavaScript** — 零依赖，无需构建
- **Web Audio API** — 程序化合成所有音效（出牌、背景音乐、爆炸等）
- **Web Speech API** — 系统内置TTS语音播报，离线可用
- **BroadcastChannel + localStorage** — 同局域网多标签页通信
- **Canvas** — 粒子背景 + 胜利彩带特效

## 📱 兼容性

- Chrome / Edge / Safari / Firefox 最新版
- 支持桌面端和移动端
- 完全离线运行，无需网络

## 📁 文件结构

```
├── index.html   # 页面结构（6个屏幕：开始/规则/大厅/房间/游戏/结算）
├── style.css    # 全部样式（响应式 + 动画）
├── game.js      # 核心游戏逻辑（牌型判断/AI/出牌/叫地主）
├── sound.js     # 音效系统（Web Audio + TTS语音）
└── network.js   # 多人联机模块（BroadcastChannel）
```
